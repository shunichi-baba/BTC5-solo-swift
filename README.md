# Installing & Setup

### ChatApp
1.  If you have not installed cocoapods, please install it.

```zh
sudo gem install cocoapods
```

2.  cocoapods setup

```zh
pod setup
```

3.  change the directory to the ChatApp

```zh
cd ChatApp
```

4.  install modules

```zh
pod install
```

5.  xcode open

6.  xcode Simulator start
